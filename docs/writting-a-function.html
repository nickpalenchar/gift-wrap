Writing a function for giftwrap

The following outlines acceptable ways to write a function for giftwrap to parse. The first section describes compatable function formats, and the second describes compatible file formats. Read both for maximum understanding!

Compatible function formats

Function declaration (named or anonymous)


function hello (name) { console.log("hello" + name) }

function (name) { console.log("hello" + name) }

Function expression (var, let, or const)

NOTE: variable names will not be mentioned in the CLI if the function name is anonymous

// will find anonymous function
var hello = function (name) { console.log("hello" + name) }

// will find function named `hello`
let helloAgain = function hello (name) { console.log("hello" + name) }

const helloOneMoreTime = function (name) { console.log("hello" + name) }

NodeJS `module.exports` expression


// ... random code ... //

module.exports = function (name) { console.log("hello" + name) }

// possibly more random code
